Tmux

tmux is a terminal multiplexer. It lets you switch easily between several programs in one terminal, detach them (they keep running in the backround) and reattach them to a different terminal.

notes

Sessions

            
              tmux new | nuova sessione
              ctrl+a d | detach
              tmux new -s session_name | crea con nome
              ctrl+a s | lista sessioni
              cd ~/Works/presentazione_vim/
              ctrl+a s | lista sessioni

Windows

            
              ctrl+a c | crea
              ctrl+a , | rinomina
              ctrl+a w | lista finestre
              ctrl+a p | precedente
              ctrl+a n | successiva
              ctrl+a 0...n | seleziona finestra n
              ctrl+a & | chiudi finestra

Esistono anche i pannelli ma non li uso

Smug

Diamo un senso a tmux

Permette di salvare un file di config per ogni sessione di tmux

Ogni file permette di iniziare una sessione con diversi layout, finestre e pannelli

Utile soprattutto per salvarsi i comandi con paramentri

Rispetto a tmuxinator ha l'opzione per le finestre manuali

Potrebbe diventare uno standard dei progetti e avere sempre un file di config nella repo

cd ~/.config/smug/ && ll nvim presentazione_vim.yml uhubs-backend.yml tool-reseller-api.yml
